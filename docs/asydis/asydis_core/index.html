Source
Expand description

shared-display-core contains the core components of [shared-display] that are required to add screen-sharing support for a display driver.

This crate should only be used by drivers that extend shared-display. Applications should instead depend on [shared-display] itself.

This crate heavily relies on and builds on top of embedded-graphics and various crates of the embassy project.
